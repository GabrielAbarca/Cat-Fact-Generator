# React Technical Interview: Cat Fact Generator

## 🚀 Key Features

- **Asynchronous Fact Fetch**: Fetches a random cat fact from the `catfact.ninja` API along with a Cat image displaying the first 3 words from the fact.
- **Dynamic Image Overlay Generation**
- **Custom Hooks Architecture**
- **Responsive Design**

## 📦 You can run the project locally!

### Prerequisites
- Node.js (v18.0.0 or higher recommended)
- npm or yarn

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Project
Start the Vite development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

---

## 🌐 External APIs Used
- **Facts API**: [catfact.ninja/fact](https://catfact.ninja/fact)
- **Image Generator API**: [cataas.com](https://cataas.com)
